### What is this?

This is a **Magisk module** to theme system apps of MIUI to match Android 12 Material You coloring scheme but keep it as Vanilla MIUI as possible.

### What's currently themed

1. Control Center
2. MIUI Dialer and Contacts
3. MIUI Messaging
4. Settings
